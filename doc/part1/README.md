## 介绍

这个教程主要是针对以下的情况:

* 你在本地有台电脑(系统任意)
* 有一台高性能服务器(ubuntu系统)可供使用

内容主要包括:

* 初次登录服务器新建一个自己的账号
* 一顿操作，让你可以在本地写程序，在服务器跑程序
* 一顿操作，开始用服务器上的用显卡跑程序(施工中)
