## 介绍

这个教程主要是写给新来实验室的童鞋,帮助大家尽快上手使用服务器

内容主要包括:

- 初次登录服务器并且给新建一个自己的账号
- 介绍刚刚接触服务器应该做的一些事
- 介绍Pycharm
- 掌握GPU加速相关软件的安装和配置
